# eMooring #

A [mooring ](https://en.wikipedia.org/wiki/Mooring_(watercraft))refers to any permanent structure to which a vessel may be secured.

## How to Use ##
First, make sure you have Node.js & MongoDB installed.
Start app with **nodemon**
